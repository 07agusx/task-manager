# Task Manager CLI

A command-line task manager built with Python as a learning project to practice Object-Oriented Programming (OOP), project organization, and file persistence.

Tasks are automatically saved to a JSON file, allowing them to persist between program executions.

---

## Features

- Add new tasks.
- Mark tasks as completed.
- Delete tasks.
- Display all tasks with their ID and completion status.
- Automatic data persistence using JSON.
- Modular project structure.
