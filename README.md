# Task Manager CLI

A simple command-line task manager built in Python. You can add, complete, and delete tasks. All data is saved automatically in a JSON file, so your tasks persist between sessions.

## Requirements

- Python 3.10 or higher (required for `match/case`)

## How to run

```bash
python taskmanager.py
```

## Options

- **Add** — Add a task by entering its name.
- **Complete** — Mark a task as completed using its unique ID.
- **Delete** — Delete a specific task using its unique ID.
- **Show task list** — Display all tasks with their ID and status.
- **Exit** — Exit the program.
